# Logos de Aliados Estratégicos

Coloca aquí los logos de las marcas aliadas.

## Formato recomendado:
- Formato: SVG (preferido) o PNG con fondo transparente
- Tamaño: Ancho máximo 300px
- Colores: Versión monocromática o color original

## Nombres de archivo:
- `apart-pipa.svg`
- `reiner.svg`
- `mundoviajesok.svg`
- `poseidon.svg`

## Actualizar en PartnersCarousel.astro
Cuando agregues los logos, actualiza las rutas en:
`src/components/ui/PartnersCarousel.astro`

Los placeholders actuales se reemplazarán automáticamente cuando agregues los archivos con los nombres correctos.

## Nota:
Si tienes logos en PNG, cámbialos a formato SVG para mejor calidad y rendimiento, o ajusta la extensión en el código:
```typescript
logo: "/images/partners/apart-pipa.png"
```
