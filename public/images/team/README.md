# Imágenes del Equipo

Coloca aquí las fotos de los miembros del equipo.

## Formato recomendado:
- **Gabriela**: 800x1066px (proporción 3:4) - imagen vertical
- **Carolina y Sebastian**: 800x800px (proporción 1:1) - imagen cuadrada
- Formato: JPG o PNG
- Tamaño máximo: 500KB por imagen (optimizado para web)

## Nombres sugeridos:
- `gabriela.jpg`
- `carolina.jpg`
- `sebastian.jpg`

## Actualizar en TeamSection.astro
Cuando agregues las imágenes, actualiza las rutas en:
`src/components/sections/TeamSection.astro`

Cambia de:
```typescript
image: "https://images.pexels.com/..."
```

A:
```typescript
image: "/images/team/gabriela.jpg"
```
